#### AuctionFlow - Backend

### Git Convention
> 1. 적절한 커밋 접두사 작성
> 2. 커밋 메시지 내용 작성

> | 접두사         | 설명                 |
> |-------------|--------------------|
> | Feat :      | 새로운 기능 구현          |
> | Add :       | 기존 기능에 간단한 추가      |
> | Fix :       | 버그 수정              |
> | Docs :      | 문서 추가 및 수정         |
> | Style :     | 스타일링 작업            |
> | Refactor :  | 코드 리팩토링 (동작 변경 없음) |
> | Test :      | 테스트                |
> | Deploy :    | 배포                 |
> | Conf :      | 빌드, 환경 설정          |
> | Chore :     | 기타 작업              |
